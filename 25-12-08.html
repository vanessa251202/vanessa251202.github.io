識別字命名符號

  • 第一個字：a-z A-Z _(底線) $
  • 第二個字開始：a-z A-Z 0-9 _(底線) $
  • Java 保留字、關鍵字不可用於命名

命名習慣

  • 類別、介面：Hello、HelloWorld
  • 方法、變數：name、employeeId
  • 常數(不可改)：PI、MAX_VALUE

程式碼範例：HelloWorld


// 單行註解

// 宣告主類別

// public 		存取修飾字：公開
// class 		宣告類別關鍵字
// HelloWorld		自訂類別名稱
// { }			程式區塊

public class HelloWorld{

    // 主方法：程式進入點 
    
    // public  		存取修飾字：公開
    // static 		類別成員，可以直接被呼叫 
    // void 		方法型別，void 表示這個方法沒有回傳值
    // main             主方法名稱(固定寫法)
    // (String[] args)  方法參數
    //                  String		字串型別
    //                  []              陣列
    //                  args            參數名稱 

    public static void main(String[] args){
	
        // 程式開始執行
        // 敘述句，要用【;】結束
        // 標準輸出，在螢幕打印資料
        System.out.println("Hello! World!");

    } // 主方法結束 

} // 主類別結束
